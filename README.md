# Online Food Ordering System

A full stack mobile food ordering application built for the SE2020 Web and Mobile Technologies group assignment.

## Tech Stack
- Frontend: React Native (Expo)
- Backend: Node.js + Express.js
- Database: MongoDB

## Project Structure
- `backend/` - Express API and database logic
- `mobile-app/` - React Native mobile application
- `docs/` - project planning documents

## Core Modules
- Authentication
- Menu Management
- Cart Management
- Order Management
- Payment Management
- Review & Rating Management
- Delivery Management

## Team Workflow
Each member should work on a separate branch and create pull requests before merging to main.
